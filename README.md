# 동래구 · 부산광역시

> 동래구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-dongnae.github.io/**

동래구는 조선시대 동래도호부가 자리했던 부산 행정·군사의 옛 중심지로, 오늘날에도 온천장과 사직 일대를 아우르는 전통과 생활의 고장입니다. '동래(東萊)'라는 지명은 동쪽의 내산(萊山), 곧 신선이 산다는 봉래산에서 비롯되었다고 전하며, 757년 신라 경덕왕 때 동래군으로 개칭된 이래 천 년 넘게 이어져 온 유서 깊은 이름입니다. 온천천이 남북으로 관통하는 이 지역에는 우리나라에서 가장 오래된 온천 가운데 하나인 동래온천이 솟아, 예부터 목욕과 요양을 겸한 휴양지로 이름났습니다. 임진왜란 동래성 전투의 항전 정신을 기리는 충렬사와 동래읍성, 가야 문화를 증언하는 복천동 고분군 등 부산의 뿌리를 담은 문화유산이 밀집해 있습니다. 사직야구장과 부산종합운동장을 품어 프로 스포츠의 열기가 넘치는 한편, 명륜·수안·안락 일대의 오래된 주거지가 안정된 정주 여건을 이루고 있습니다.
